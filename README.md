# Nodejs Export Array to excel 
> This is a simple example of how to export array to excel using nodejs and xlssx module.

## Installation
```bash
npm install
```
## Setup Directory
```js
    ...
    const OUTPUT = './output'
    const FILENAME = 'employees.xls';
    ...
```

## Usage
```bash
npm start
```
